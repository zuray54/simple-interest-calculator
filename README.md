# Simple Interest Calculator

A simple program that calculates simple interest based on principal, rate, and time.

## Description
Calculates simple interest using the formula:

**SI = (P × R × T) / 100**

- P = Principal amount
- R = Rate of interest (per annum)
- T = Time (in years)

## How to Run
1. Clone this repository
2. Run `python simple_interest.py`
3. Enter principal, rate, and time when prompted
